動画抽出モード UI サンプル (v2)

再生画面下部の改修サンプル。 通常は 保管 / 削除 / サムネイル / 抽出 の 4 ボタンだけが並びます。 抽出 をクリックすると mode 切替で「保管 / 削除 / サムネイル」 が非活性化し、 専用エリアが滑らかに展開して 動画抽出 (IN/OUT/動画保存) と 画像抽出 (画像保存) の 2 行が現れます。 シークバー上の まち針 は抽出 mode 中だけ表示&ドラッグ可能。

OFF: イベント時刻なし → 抽出モード ON で IN=0% / OUT=100% (= 動画全体) が default

動作の試し方

  • 抽出 ボタンを押すと抽出モードに切替わる。 もう一度押すと終了。 トグルボタン自身の色変化 (= 緑 → 赤) で現在のモードがわかる。
  • 抽出モード ON で IN/OUT が未設定なら、 自動で default 範囲を設定:
    • イベント時刻あり → IN = eventTime − skipSeconds、 OUT = eventTime + skipSeconds (= E キー位置から ± skipSeconds の幅)
    • イベント時刻なし → IN = 0、 OUT = 動画末尾 (= 動画全体)
    上のチェックボックスで切替えて両ケースを試せます。
  • 抽出モード中はシークバー上の まち針 (= 緑) を頭の丸い部分でつまんで左右にドラッグして IN / OUT を動かせる。 通常モードでは表示されない。
  • IN は OUT の左、 OUT は IN の右までしか移動できない (= 反転防止)。
  • 「IN」 / 「OUT」 ボタンは現在位置をマーカーにセット (= 既存の挙動)。
  • 赤い は イベント時刻マーカー (= 既存仕様、 抽出モードでも表示)。

UI 構成 (= 状態別)

  • 通常モード: 📦 保管 🗑 削除 🖼 サムネイル ... ✂ 抽出 の 4 ボタン。 「保管 / 削除」 はアイコンのみで tooltip 表示、 「サムネイル」 は tooltip で 「イベント時刻を設定し、サムネイルを差し替える」。
  • 抽出モード: 上の 3 ボタンが非活性 (= 灰色化)、 「抽出」 ボタンは終了用に色変化。 下に 2 行が展開:
    • 動画抽出: ◀ IN OUT ▶ ⏱ 範囲長 💾 動画保存
    • 画像抽出: 📷 画像保存
